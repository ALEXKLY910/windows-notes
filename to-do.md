Enable EXPO/A-XMP RAM profile (7400 MHz) after the installation and check out how stable it'll be.
"After Windows is installed, if you want to enable one, don’t start with 7400 unless you enjoy random crashes you’ll blame on Windows. Start with 7000 or 7200, then stress test in Windows (MemTest86, OCCT memory, or TM5/Anta777 if you’re serious). If it’s stable, great. If not, you back it down. That’s how memory overclocking works: negotiation with physics."
