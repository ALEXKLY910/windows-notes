`#` = SUPER.
`!` = Alt.
`^` = CTRL.
`+` = SHIFT.

Example: 
`#q::Run "wt.exe"`

Or

#q::{
    Run "wt.exe"
    Sleep 100
    Send "echo hi{Enter}"
}

Comments are preceded with `;`.

The file extension is `.ahk`.

It is executable and once executed the defined inside keybindings are applied.

You can put it into the Startup Folder so that it is executed on startup.
